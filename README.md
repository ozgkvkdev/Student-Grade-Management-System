# 🎓 Öğrenci Not Yönetim Sistemi

Python programlama dili kullanılarak geliştirilen, terminal tabanlı bir **Öğrenci Not Yönetim Sistemi**. Bu proje, Python'un temel konularını uygulamalı olarak pekiştirmek amacıyla hazırlanmıştır.

---

## 📖 Proje Hakkında

Bu uygulama sayesinde kullanıcı;

- 👨‍🎓 Öğrenci ekleyebilir.
- 📋 Kayıtlı öğrencileri listeleyebilir.
- 📝 Öğrencilere not girebilir.
- 📊 Not ortalamalarını hesaplayabilir.
- 🔍 Öğrenci arayabilir.
- 🗑️ Öğrenci kaydını silebilir.
- 🚪 Programdan güvenli şekilde çıkış yapabilir.

---

## 🛠️ Kullanılan Teknolojiler

- Python 3
